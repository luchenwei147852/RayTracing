# RayTracing
This is a project of a ray tracer that be able to handle opaque surfaces with lighting and shadows.
##The input type is like following:
>amb: 0.3 0.3 0.3
>sphere
>pos: 0.0 0.0 -3.0
>rad: 1
>dif: 0.3 0.3 0.3
>spe: 0.5 0.5 0.5
>shi: 1
>light
>pos: 0 0 0
>col: 1 1 1   
>...
